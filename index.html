Xtraktor

Browser-only PDF medication extraction prototype with OCR fallback, rules extraction, Chrome Built-in AI/WebLLM paths, evidence view, exports, and sample-PDF test dataset.

Status

The full single-file app and sample PDFs were generated locally. This repository has been initialized through the GitHub connector with source documentation and project metadata.

The GitHub connector used here accepts text file writes reliably, but not direct binary PDF upload from local files. Push the prepared local repo ZIP with normal Git to include the full HTML and sample PDFs.
